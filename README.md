# 北方民族大学轮滑社官方网站

## 项目介绍
这是北方民族大学轮滑社的官方网站，用于展示社团风采、教师信息、活动安排和招新报名。

## 网站功能
- 轮滑社介绍
- 教师风采展示
- 社团活动信息
- 精彩瞬间相册
- 在线报名系统

## 访问地址
- 在线演示：https://nmu-lunhua-club.github.io
- 备用地址：https://numlunhua.vercel.app

## 技术栈
- HTML5 + CSS3
- JavaScript（原生）
- 响应式设计

## 如何部署
1. 克隆本项目
2. 将 `lunhua.html` 重命名为 `index.html`
3. 上传到 GitHub Pages 或 Vercel
